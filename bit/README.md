## 异或

异或运算就是无进位相加
> 0 ^ N = N
>
> N ^ N = 0
>
> a ^ b = b ^ a [交换律]
>
> a^(b^c) = (a^b)^c [同样一批数不论以什么顺序进行异或，最终的结果一定都相同]
